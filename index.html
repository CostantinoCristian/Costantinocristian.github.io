Experiencia Laboral

VASS Oct 2023 – Jun 2024

En VASS, trabajé como Desarrollador Java enfocado en la plataforma Magnolia. Durante este tiempo, desarrollé plantillas y componentes personalizados utilizando, además de Java, herramientas como FreeMarker, YAML, HTML y JavaScript. Además, para desarrollar los componentes, creaba consultas SQL optimizadas, asegurando un acceso eficiente a los datos y manteniendo el rendimiento del sistema.

Como entorno de desarrollo, utilicé Eclipse y Visual Studio Code, junto con Maven para la gestión de dependencias, Apache para facilitar la implementación de los proyectos y GitHub para un correcto control de versiones.

Mi trabajo también incluyó la integración de sistemas externos que optimizaron la funcionalidad de la plataforma, y de la migración de contenido desde otras plataformas, asegurando siempre la integridad de los datos.

Participé activamente en la detección y resolución de fallos, realizando pruebas de calidad para garantizar el correcto funcionamiento y un rendimiento óptimo.
