Proyectos destacados

En esta sección, presento una selección de mis proyectos más representativos. Aunque tengo una gran cantidad de programas creados, he decidido compartir solo aquellos que están completos y totalmente funcionales.

Además, estos proyectos no fueron realizados para terceros ni empresas, lo que me ha permitido experimentar y aprender de manera más creativa y autónoma.

TFC DAM - Author's Death

"Authors Death" es mi proyecto final de curso de la FP DAM se trata de una aplicación servidor que permite a los usuarios escribir libros y compartirlos con la comunidad. Los usuarios pueden leer los libros de otros y dejar comentarios, fomentando la interacción entre escritores y lectores dentro de la plataforma.

El proyecto fue desarrollado en Java utilizando el framework Spring, creando una serie de APIs que gestionan la interacción con una base de datos MySQL. Estas APIs manejan funcionalidades clave como la creación de usuarios, el inicio de sesión, la creación de libros, la lectura y los comentarios, además de controlar la visualización de los libros disponibles para leer.

Aunque las APIs podían probarse usando Postman, también desarrollé una web básica con una maquetación sencilla que funcionaba como entorno de pruebas. Esto me permitió verificar rápidamente que todo funcionara correctamente y optimizar la velocidad de las interacciones.

Horóscopo

Este proyecto utiliza la manipulación del DOM para generar dinámicamente el código HTML desde JavaScript.

La web presenta una lista de los signos zodiacales y ofrece descripciones detalladas de cada uno. Además, incluye una calculadora que permite a los usuarios determinar su signo zodiacal según su fecha de nacimiento.

Actualmente, la información sobre los signos se obtiene a partir de archivos XML estáticos, pero es posible implementar un servidor que permita actualizaciones periódicas, asegurando que el contenido del horóscopo se mantenga fresco y relevante.
