Proyectos destacados

Te invito a conocer algunos de los proyectos que realicé de manera autónoma. Aunque tengo una gran cantidad de programas creados, comparto aquí los que están completamente funcionales.

TFC DAM - Author's Death

Author's Death es mi proyecto final de la Formación Profesional DAM. Se trata de una aplicación servidor que permite a los usuarios escribir libros y compartirlos con la comunidad. Los usuarios pueden leer los libros de otros y dejar comentarios, lo que fomenta la interacción en la plataforma.

El proyecto fue desarrollado en Java utilizando el framework Spring, donde implementé APIs para gestionar la interacción con una base de datos MySQL. Estas APIs manejan todas las operaciones de la aplicacion como son la creación de usuarios, el inicio de sesión, la creación de libros, su lectura y comentarios.

Aunque las APIs podían probarse usando Postman, también desarrollé una web básica con una maquetación sencilla que funcionaba como entorno de pruebas.

Horóscopo

Este proyecto utiliza la manipulación del DOM para generar dinámicamente el código HTML desde JavaScript.

La web presenta una lista de los signos zodiacales y ofrece descripciones de cada uno. Además, incluye una calculadora que permite a los usuarios determinar su signo zodiacal según su fecha de nacimiento.

Actualmente, la información sobre los signos se obtiene a partir de archivos XML estáticos, pero es posible implementar un servidor que permita actualizaciones periódicas.

Tres en línea

Un pequeño juego para dispositivos móviles desarrollado en Android Studio.

Se trata del clásico Tres en Línea (TA TE TI o Tic Tac Toe), creado como ejercicio práctico para programar en Android y profundizar en sus interfaces.
